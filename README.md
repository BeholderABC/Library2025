# 图书馆信息管理系统（WebLibrary）

## 项目简介
本项目是一个基于ASP.NET Core Razor Pages的图书馆信息管理系统，旨在构建一个功能完善、操作便捷的在线图书馆管理平台，除基础的用户注册、登录、图书检索、借阅、归还、预约等外，还支持图书评分、评论，图书推荐和文化活动宣传等功能，同时为管理员提供图书管理、用户管理、借阅统计等后台管理能力。
