# 图书馆信息管理系统（WebLibrary）

## 项目简介
本项目是一个基于ASP.NET Core Razor Pages的图书馆信息管理系统，可以通过网页方便地管理图书信息。

## 初始功能
- 图书信息管理：
  - 查看图书列表
  - 添加新图书
  - 删除图书
....
## 使用方法
1. 启动项目：
   - 使用Visual Studio启动项目，自动在浏览器中打开页面
2. 在导航栏点击"图书管理"进入预约管理草稿图样：
## 目录结构说明
- Pages/Book/：图书管理相关页面和代码
- wwwroot/：静态资源（样式、脚本等）
- Program.cs：项目入口
- WebLibrary.csproj：项目配置
...
